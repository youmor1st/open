# Политика конфиденциальности и безопасности SafeLearn

## Принципы
- Минимальный сбор данных
- Не храним персональные данные без необходимости
- HTTPS для передачи данных
- Планируем использовать шифрование at-rest

## GDPR
Мы следуем лучшим практикам GDPR:
- Прозрачность
- Контроль пользователя над данными
- Отзыв согласия
