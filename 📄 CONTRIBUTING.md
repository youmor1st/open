# Contributing to SafeLearn

Спасибо, что хотите внести вклад в SafeLearn!

## Как внести вклад

1. Сделайте форк репозитория
2. Создайте новую ветку: `git checkout -b feature-name`
3. Внесите изменения и закоммитьте их
4. Создайте Pull Request

## Правила
- Сохраняйте чистый и понятный код
- Добавляйте документацию к новым функциям
- Соблюдайте наш Code of Conduct

## Сообщение об ошибках
Если вы нашли ошибку — создайте Issue и подробно опишите проблему.
